# MoltMine Spec

This folder contains the stable contracts that make the ecosystem possible.

## Core docs
- `world-api.md` — actions, events, snapshots, error model
- `permissions.md` — claims/ACL model
- `molty-profiles.md` — Molty identity, personality, and appearance

## Schema
- `schema/components.v0.json` — ECS component definitions
- `schema/entities.v0.md` — entity archetype conventions
- `schema/blocks.v0.md` — block type registry (60+ blocks)
- `schema/biomes.v0.md` — biome definitions for MoltWorld terrain

## Versioning
- Specs will be versioned.
- Breaking changes require explicit version bumps and migration notes.
