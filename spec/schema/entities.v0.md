# Entities v0 (Draft)

This document defines a *starter* set of entity archetypes.

## Archetypes

### Avatar
Components:
- Transform
- Appearance
- (optional) Ownership

### Prop (static)
Components:
- Transform
- Appearance
- (optional) Ownership

### Claim (region)
Components:
- Transform (anchor)
- ClaimData (TBD)

## Notes
- Archetypes are not enforced types; they are conventions.
- The authoritative server may choose storage/serialization formats.
